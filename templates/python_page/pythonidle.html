    Introducción al IDLE de Python


    En la mayoría de estas guías estaremos empleando el IDLE que viene con Python, para no complicar las cosas demasiado.

    Primeramente, tenemos que comprender la teoría de qué es un IDLE: Un IDLE es un entorno de desarrollo integrado para Python, que se incluye con la implementación predeterminada del lenguaje desde la versión 1.5.2b. Ahora que sabemos lo que es, vamos a observar lo que viene siendo el IDLE de Python:




    Lo que podemos ver ahora mismo es la consola del IDLE Python, el inicio del input (donde vamos a escribir) está marcada por el >>>, esto significa que cualquier comando que introduzcamos será ejecutado inmediatamente como cualquier consola de comandos.


    Esto puede resultar muy práctico especialmente cuando estamos trabajando con ciertas funciones para las cuales necesitamos este tipo de entorno, pero no es práctico para lo que viene siendo escribir programas largos. Para escribir código de forma más limpia lo que podemos hacer es crear un nuevo archivo:
