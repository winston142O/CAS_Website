    ¿Cómo empezar con Python?


    Antes de que aprendas a utilizar el lenguaje, necesitas instalarlo y configurarlo.

    Primero, para instalar Python debemos ir al sitio web oficial de Python para descargarlo, y elegir una versión adecuada para nosotros.




    Al elegir una versión de Python, recomendamos elegir la versión más reciente, pero si deseas podrías investigar para ver cual te conviene más, ya que diferentes versiones a veces pueden funcionar diferente, de igual forma, para los principiantes simplemente recomendamos la versión más reciente.




    Luego de abrir el instalador, este practicamente se explica por si mismo, simplemente asegúrate de marcar ambas casilas. Luego de que se complete la instalación, valga la redundancia, estás listo para aprender a usar Python.


    ¿Cómo saber si se instaló correctamente?


    Obviamente, una vez terminada la instalción querremos saber si se instaló correctamente o no. Para ver esto, debemos seguir los siguientes pasos:


    • 1. Presionar la tecla 'windows' y 'r' al mismo tiempo.

    • 2. Escribir "cmd" en el campo.


    • 3. Presiona "enter".


    • 4. Escribe "python --version" y presiona "enter"


    Listo, como podemos ver, Python está instalado, y nos dice qué versión instalamos.


    Recomendaciones


    En nuestra guías, especialmente las primeras, serán principalmente en el editor IDLE de Python normal, pero es bueno familiarizarnos con otros interpreters... Estos son los que recomendamos:


    • Visual Estudio

    • IDLE Python

    • Notepad++

    Visual Studio Code: Recomendamos VS Code porque es uno de los editores más completos disponibles, no solamente para Python pero para cualquier lenguaje. Es organizado, tiene muy buenas funciones y opciones, es personalizable y fácil de tomar, y sobre todo nos ofrece gran comomidad al manejar grandes directorios, como por ejemplo el de un website. Sin duda recomendamos VS Code para la mayoría de personas.


    Python: El editor de Python es simplemente eso, un simple editor que viene integrado con Python, como si fuera un bloque de notas, funciona bastante bien.


    Notepad++: Notepad++ es literalmente bloque de notas con esteroides, pero aún así es el editor que menos recomendamos para principiantes, puede resultar un poco difícil a veces o poco intuitivo.
