    Guía para publicaciones.

    Antes de hacer una discusión o pregunta en nuestro foro ( o responder ), deberías observar detalladamente estas pautas a seguir para publicar en el foro.

    • Antes de preguntar, trata de investigar un poco por tu cuenta, puede ser que tu pregunta ya haya sido preguntada o incluso respondida.

    • En caso de hacer cualquier tipo de comentario en una publicación existente, se amable, no está permitido ningún tipo de insulto. Debemos ser una comunidad amigable para tanto como nuevos programadores y programadores experienciados.

    • Asegúrate de incluir todos los detalles que puedas en tu pregunta, mientras más detalles sobre tu duda o situación proporciones a la comunidad, más fácil será responderte para las persona que deseen ayudarte.

    • Si no te sabes la respuesta a una pregunta, no hagas comentarios del tipo "yo tampoco sé", en su lugar, intenta contactar con el creador original de la pregunta y pregúntale mediante mensaje directo si pudo encontrar una solución.

    • Se amable, y ayuda a la comunidad a crecer.

    Guía para darle formato a tu publicación.

    Tenemos varias opciones para aplicar formato a tu texto

    • Utiliza # para aplicar formato de

      Título.


    • Utiliza * alrededor de tu texto para italics.

    • Utiliza ** alrededor de tu texto para negritas.

    • Utiliza ``` y ``` alrededor de tu texto para dar formato de código.

    • Utiliza ``` y ``` arriba y abajo de tu texto para dar formato de bloque de código.

    • Utiliza > antes de tu texto para dar formato de cita o resaltar algo una cita.

    • También puedes publicar imágenes adicionales dentro de tu post arrástrandolas al cuadro de texto.

    • Puedes crear listas numeradas utilizando 1. y * para crear listas de viñetas.

    • Utiliza el modelo [palabra o frase](URL) para crear hipervínculos.
      Ejemplo:
      [hipervínculo](http://localhost:8000/forum/post/new/) = hipervínculo
